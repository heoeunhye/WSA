# 미디어 쿼리

기술의 발달로 웹이 표시되는 매체가 매우 빠르게 늘어나고 있습니다. 아직까지 웹이 가장 많이 사용되는 곳은 PC이겠지만, CSS는 기존에도 웹을 다양한 매체에 표시하기 위한 기능을 가지고 있었으며, CSS3 에서는 그 기능이 크게 강화되었습니다. 이번에는 웹을 다양한 매체에 선택적으로 적용할 수 있는 방법인 Media Query에 대해 살펴보겠습니다.

# Media Query의 사용

Media Query는 CSS3에서 새롭게 추가된 속성이라기보다는 CSS2.1의 미디어 기능이 확장된 것으로 볼 수 있습니다. HTML4.01과 CSS2.1에서는 "aural(청각)", "braille(점자)", "handheld(휴대 기기)", "print(인쇄)", "projection(프로젝터)", "screen(화면)", "tty(전신 인자기)", "tv(텔레비전)", "embossed(양각)", "speech(음성)", "all(모든 미디어)"’의 미디어 타입을 지정함으로써 사용자 환경에 따라 CSS를 선택적으로 적용할 수 있도록 정의하고 있습니다.

CSS의 선택적 적용은 다음과 같은 방법으로 지정할 수 있습니다.

📌link 요소의 media 속성을 사용하여 미디어 타입에 따라 선택적으로 CSS를 적용하는 방법

<link rel="stylesheet" type="text/css" media="screen" href="screen.css">
<link rel="stylesheet" type="text/css" media="print" href="print.css">
1
2

해당 코드에서는 screen.css와 print.css 2개의 CSS 파일을 선택적으로 적용하기 위해 media 속성을 이용하였습니다.

📌style 요소의 media 속성을 사용하여 미디어 타입에 따라 선택적으로 CSS를 적용하는 방법

<style type="text/css" media="screen">
/* 미디어 타입이 screen일 때 적용되는 CSS */
h1 {
 color : #ff0000 ;
}
p {
 font-size :1.2em ;
}
</style>
<style type="text/css" media="print">
/* 미디어 타입이 print 때 적용되는 CSS */
h1, p {
 color : #000000 ;
 font-size : 1em ;
}
</style>

📌@media 규칙을 사용하여 미디어 타입에 따라 선택적으로 CSS를 적용하는 방법

@media 규칙은 하나의 CSS 파일 내에서 특정 매체에만 지정할 CSS 속성 구간을 작성하는 방법으로 다음과 같이 선언할 수 있습니다.

/* 모든 매체에 공통적으로 적용되는 CSS */
h1 {
 color: #ff0000;
}
p {
 font-size: 1.2em;
}

@media print {
/* 미디어 타입이 print일 때 적용되는 CSS */
 h1, p {
  color: #000000;
  font-size: 1em;
 }
}

# Media Query의 변화와 특징

CSS2.1까지는 단순히 매체(media)의 종류(type)에 따른 구분만 가능했지만, CSS3에서는 해당 매체의 특성에 따라 구분할 수 있는 기능이 추가되었습니다. 특히 이 방법은 매체의 해상도에 따른 구분으로, 사이즈가 다른 여러 단말에 적용할 CSS를 분리하여 단말에 따른 다양한 사용자 경험을 제공할 수 있습니다.

다음은 Media Query에서 매체의 해상도나 특징을 구분할 수 있는 방법입니다.

📌width

width는 출력 매체의 너비의 따라 미디어 유형을 정의하는 기능으로, 지정 너비(width), 최소 너비(minwidth), 최대 너비(max-width)로 사용할 수 있습니다. width는 매체의 너비가 지정된 너비와 같을 경우 해당 CSS를 적용하고, min-width나 max-width는 매체의 너비가 지정된 너비보다 클 경우 또는 작을 경우 해당 CSS를 적용합니다.

<!--link 요소의 사용 -->
<link rel="stylesheet" media="screen and (min-width:640px)" href="style.css">

<!--style 요소의 사용 -->
<style type="text/css" media="screen and (min-width:640px)">
 /* CSS 코드 */
</style>

# Media Query의 조건식

Media Query는 여러 가지 조건식을 지정하여 사용할 수 있으며, 각각의 조건식은 다음과 같은 특징과 형식을 가지고 있습니다.

📌not / only

not과 only는 미디어 쿼리의 조건을 제한하거나 부정하는 접두사입니다.

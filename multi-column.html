# 다단 관련 속성

웹 콘텐츠는 기본적으로 텍스트로 이루어졌지만, 인쇄물에서 자주 사용되는 다단(Multi Column) 구조를 앞서 살펴본 속성들만으로는 구현하기가 어렵습니다. CSS3에서는 다단 문서를 쉽게 만들 수 있는 Multiple column layout 모듈이 추가되었습니다. 이번에는 CSS3에 추가된 다단 속성을 살펴보겠습니다.

# column–width 속성

column-width는 다단 구성 시 단의 너비를 지정하는 속성입니다. 기본 값은 auto이며, 단의 너비를 자동으로 계산합니다. column-width 속성 값을 auto로 지정한 후에 다단을 구성하기 위해서는 column-count 속성이 필요합니다.

auto 이외에 단의 너비를 길이 값으로 지정할 수 있습니다. 이때 단의 개수는 요소의 너비에 따라 자동으로 계산됩니다. 여기서 지정된 값은 정확한 길이 값이 아니라 상위 요소가 단을 구성하는 최소 기준 값이됩니다. 만약 요소의 너비가 500px일 때 column-width 속성 값을 200px로 지정하면 200px보다 큰 2개의 단이 요소 500px 너비를 모두 채우게 되고, 요소의 너비가 더 늘어나서 200px짜리 단을 추가할 수 있을 만큼 넓어지면 단이 하나 더 추가되면서 단의 폭이 200px까지 줄어들게 됩니다.

/* 예시 */
div { colum-width : 200px ; }

# column–count 속성

column-count는 다단 구성 시 단의 개수를 지정하는 속성입니다. 속성 값에는 단위가 없는 0이 아닌 정수 값으로 단의 개수를 지정합니다. 이때 생성되는 단의 너비는 자동으로 계산됩니다. column-count 속성으로 다단을 구성했을 경우에는 요소의 너비가 증가하더라도 단의 개수가 유지되며, 단의 너비만 더 넓어집니다.

column-count 속성의 기본 값인 auto는 단의 개수를 자동으로 지정합니다. column-count 속성 값을 "auto"로 지정하고, 다단을 구성하기 위해서는 column-width 속성을 지정해야 합니다.

/* 예시 */
div { column-count : 3 ; }

# columns는 속성

columns는 단의 너비와 개수를 일괄 지정하기 위한 대표 속성입니다. columns 속성에 지정할 속성 값은 column-width 속성, column-count 속성과 동일합니다.

/* 예시 */
div { columns : 200px 3 ; }

# column-gap 속성

column-gap은 단과 단 사이의 간격을 지정하기 위한 속성으로, 길이 단위 값을 이용하여 단의 간격을 지정할 수 있으며, 길이 값은 0보다 작을 수 없습니다. 그리고 normal 속성 값의 경우, 아직 스펙상으로 값이 명확히 결정되지 않았으며, 현재 "1em"이 제안된 상태입니다.

/* 예시 */
div { column-gap : 50px ; }
1
2

# column-rule 속성

column-rule은 단과 단 사이의 구분선을 표현하기 위한 속성입니다.

column-rule 속성은 border 속성과 같은 단축 속성으로, 다음과 같은 세부 속성을 한꺼번에 지정할 수 있습니다.

  • column–rule–width : 다단 구분선의 굵기를 지정합니다. border-width 속성과 동일하게 thin, medium, tick, 길이 단위 값으로 속성 값을 지정할 수 있습니다.
  • column–rule–style : 다단 구분선의 형태를 지정합니다. border-style 속성과 동일하게 none, hidden, dotted, dashed, solid double, groove, ridge, inset, outset으로 속성 값을 지정할 수 있습니다.
  • column–rule–color : 다단 구분선의 색상을 색상 값으로 지정합니다.
/* 예시 */
div { column-rule : #f00 solid 2px ; }

# column-span 속성

column-span은 다단 구성 시 여러 단을 차지하는 요소를 지정하기 위한 속성으로, none 값은 해당 요소가 여러 단을 차지하지 않도록 하는 것을 의미하고, all 값은 지정된 요소가 여러 개의 단 영역을 차지하도록 표현할 때 사용합니다.

/* 예시 */
div { column-span : all ; }
1
2

# column-fill 속성

column-fill은 다단 영역 콘텐츠 흐름의 영향을 받는 방식을 지정하는 속성입니다. 방식은 크게 auto와 balance가 있습니다. auto는 기본 값으로, 단의 길이가 다른 단의 길이에 따라 서로 다르게 표현될 수 있으며 단이 순서대로 채워집니다. 두 번째로 balance는 단의 높이가 다른 단의 크기에 따라 균형을 이루도록 단이 순서대로 채워집니다.

/* 예시 */
div { column-fill : balance ; }

# break–before, break–after, break–inside 속성

break-before, break-after, break-inside는 다단의 분할 방식을 지정하는 속성입니다.

break 관련 속성 값에는 다음과 같은 분할 방식이 적용됩니다.

  • auto : 페이지나 다단 나누기를 강제하거나 금지하지 않습니다.
  • always : 항상 요소 앞에서 페이지나 단을 분할합니다.
  • avoid : 요소 앞에서 페이지나 단을 분할하지 않습니다.
  • left : 요소 앞에서 페이지나 단을 분할하고, 다음에 오는 단이 왼쪽에 배치되도록 합니다.
  • right : 요소 앞에서 페이지나 단을 분할하고, 다음에 오는 단이 오른쪽에 배치되도록 합니다.
  • page : 요소 앞에서 항상 페이지를 분할합니다.
  • column : 요소 앞에서 항상 단을 분할합니다.
  • avoid–page : 요소 앞에서 페이지를 분할하지 않습니다.
  • avoid–column : 요소 앞에서 단을 나누지 않습니다.
/* 예시 */
div { break-before : always ; }
div { break-after : column ; }
div { break-inside : avoid ; }

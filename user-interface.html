# content 속성

content 속성은 가상 요소 선택자인 :before나 :after를 선택자로 지정한 후, 특정 요소 앞이나 뒤에 콘텐츠를 추가할 수 있도록 하는 기능입니다. content 속성에는 텍스트를 포함하여 이미지나 특정 규칙에 따른 값을 지정할 수 있습니다.

/* 예시 */
body {
 counter-reset: chapter;
}
h1:before {
 content: "Chapter " counter(chapter) ". ";
 counter-increment: chapter;
}
h1 {
 counter-reset: section;
}
h2:before {
 content: counter(chapter) "." counter(section) " ";
 counter-increment: section;
}

# box-sizing 속성

box-sizing은 요소 박스의 크기를 결정하는 방식을 지정하는 속성입니다. 앞서 살펴본 박스 모델에 따라 박스의 실제 크기는 margin과 border 그리고 padding 값에 콘텐츠의 크기를 더해 결정됩니다. 그러나 box-sizing 속성을 사용하면 박스의 실제 크기를 결정하는 방식을 다르게 지정할 수 있습니다.

콘텐츠 너비는 200px, 안쪽 여백은 40px, 테두리 굵기가 10px로 지정된 요소 박스가 있다고 가정할 때, 해당 박스의 실제 크기는 기본 값인 content-box로 결정됩니다.

content-box는 요소 박스에 지정된 모든 값을 더해 박스의 크기를 결정하는 방식입니다. 즉, 콘텐츠 너비인 200px과 안쪽 여백 크기인 40px, 그리고 테두리 굵기인 10px 값을 모두 더한 값인 250px이 실제 박스의 크기가 됩니다.

그러나 border-box를 지정하면 콘텐츠에 지정된 너비인 200px에 안쪽 여백인 40px과 테두리 굵기인 10px이 포함됩니다. 그렇게 되면 실제 박스의 크기는 250px이 아니라 200px이 됩니다.

❇️padding-box의 경우 width 값에 padding 값을 포함하여 박스의 크기를 결정하는 방식이지만, 추후 해당 속성 값은 명세에서 제외될 가능성이 있습니다.

/* 예시 */
div{
 width : 200px ;
 padding : 20px ;
 border : 5px solid #000 ;
 margin : 50px 0 ;
 box-sizing : border-box ;
}

# outline 속성

outline은 outline-width, outline-style, outline-color 속성을 일괄적으로 지정하는 대표 속성입니다. border 속성과는 별개의 속성으로 요소의 외곽선을 표시하는 속성입니다. 이때 outline-width는 외곽선의 굵기, outline-style은 외곽선의 모양, outline-color는 외곽선의 색상을 지정하는 기능입니다. 이때 outline-width 값으로 지정한 크기는 요소의 크기에 포함되는 border-width 속성과 달리 요소의 크 기에 포함되지 않습니다.

/* 예시 */
div{
 outline-width : 10px ;
 outline-style : double ;
 outline-color : rgba(255, 0, 0, 0.5 ) ;
}
div{
 outline : 10px dashed rgba(0, 0, 255, 0.5 ) ;
}

# outline–offset 속성

outline-offset은 요소의 바깥쪽 영역과 외곽선 사이의 간격을 지정하는 속성입니다. 속성 값으로 음수 값도 지정할 수 있으며, 음수 값을 지정할 경우 요소의 안쪽으로 외곽선이 표시됩니다.

/* 예시 */
div{
 outline : 10px dashed rgba(255, 0, 0, 0.5 ) ;
 outline-offset : 20px ;
}

# resize 속성

resize는 사용자에 의해 요소 박스의 크기를 조정할 것인지의 여부를 지정할 수 있는 속성입니다. 이때 resize 속성을 적용하기 위해서는 overflow 속성이 지정되어 있어야 합니다. resize 속성을 사용하여 박스의 가로 크기 조절을 허용하려면 horizontal을, 세로 크기 조절을 허용하려면 vertical을, 양방향 크기 모두 조절을 허용하려면 both를 지정해야 합니다.

/* 예시 */
div {
 width : 200px ;
 height : 100px ;
 padding : 15px;
 overflow : auto ;
 resize : both ;
}

# text-overflow 속성

text-overflow는 요소 박스의 너비가 고정되어 있고, 요소 박스 내 텍스트가 박스의 크기를 넘는 경우 overflow된 텍스트를 처리하는 방식을 지정하는 속성입니다. 속성 값으로 clip과 ellipsis가 있으며, clip은 overflow된 텍스트를 잘라내고, ellipsis는 overflow된 텍스트를 숨기고 뒤에 말줄임(...) 표시를 나타냅니다.

이때 주의해야 할 점은 text-overflow 속성을 지정하려면 요소 박스의 너비가 고정값으로 지정되어야 하고 줄바꿈이 되지 않도록 white-space 속성 값을 nowrap으로 지정해야 한다는 것입니다.

/* 예시 */
div {
 width : 200px ;
 white-space : nowrap ;
 overflow : hidden ;
 text-overflow : ellipsis ;
}

# cursor 속성

cursor는 HTML 요소 위에 마우스가 오버될 때 마우스 포인터의 모양을 지정하는 속성입니다.

cursor로 지정되는 포인터는 성격에 따라 다음과 같이 분류할 수 있습니다.

  • image cursors : 지정된 이미지를 마우스 커서로 표시합니다.
  • general purpose cursors : 일반적인 포인터 형태로, "auto", "default", "none"이 general purpose cursors에 속합니다.
  • link and status cursors : 링크나 기능의 상태에 따른 포인터 형태로, "context-menu", "help", "pointer", "progress","wait"가 link and status cursors에 속합니다.
  • selection cursors : 선택에 관련된 포인터 형태로, "cell", "crosshair", "text", "vertical-text"가 selection cursors에 속합니다.
  • drag and drop cursors : drag and drop에 관련된 포인터 형태로, "alias", "copy", "move", "no-drop", "not-allowed"가 drag and drop cursors에 속합니다.
  • resizing and scrolling cursors : 요소의 크기 변경과 스크롤 관련된 포인터 형태로, "e-resize", "n-resize", "neresize", nw-resize", "s-resize", "se-resize", "sw-resize", "w-resize", "ew-resize", "ns-resize", "nesw-resize","nwse-resize", "col-resize", "row-resize", "all-scroll"이 resizing and scrolling cursors에 속합니다.
  • zooming cursors : 요소의 확대 축소와 관련된 포인터의 형태로, "zoom-in"과 "zoom-out"이 zooming cursors에 속합니다.

cursors의 형태는 CSS의 기능이 아니라 사용자의 에이전트가 지원하는 형태를 선택하는 것으로, 사용자 환경에 따라 CSS에서 정의한 cursors의 형태가 모두 지원되지 않을 수도 있습니다.

/* 예시 */
div {
 cursor : pointer ;
}

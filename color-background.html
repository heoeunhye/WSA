# 색상 및 배경 관련 속성

CSS에서 요소 박스에 색상과 배경을 지정함으로써 HTML 문서에 디자인을 할 수 있습니다. 그러므로 앞서 살펴본 박스 모델과 함께 색상 및 배경 관련 속성을 이해하는 것은 CSS를 익히는 데 있어 기본이자 중요한 부분이라 할 수 있습니다.

CSS를 이용하여 웹 페이지에 디자인을 적용할 때 가장 많이 사용되는 기능인 색상 및 배경 관련 속성은 CSS3에서 유용한 새로운 기능이 다수 추가되어 CSS를 통한 디자인의 자유도가 더욱 높아졌습니다. 이번에는 CSS3에 추가된 기능을 포함하여 색상 및 배경에 관련된 속성을 살펴보겠습니다.

# color 속성

color는 요소의 전경색을 지정하는 속성입니다. 앞에서 살펴본 바와 같이 색상과 관련된 값은 CSS3에서 기존 색상 값인 색의 3원색, 즉 빨강(Red), 초록(Green), 파랑(Blue) 값으로 표현하는 RGB 형식뿐만 아니라 색상(hue), 채도(saturation), 명도(lightness)를 의미하는 HSL 형식과 함께 투명도(alpha) 등도 제공되기 때문에 좀 더 다양한 방법으로 지정할 수 있게 되어 CSS가 표현할 수 있는 색상의 폭이 더욱 넓어졌습니다

/* 예시 */
div { color : rgb(255,0,0) ; }
div { color : rgba(255,0,0,0.5) ; }
div { color : hsl(27,88%,58%) ; }
div { color : hsla(27,88%,58%,0.7) ; }

# opacity 속성

opacity는 CSS3에 새롭게 추가된 속성으로 요소 박스의 투명도를 지정하는 속성입니다. opacity 속성 값은 "0"에서 "1" 사이의 값을 가지며, "0"은 투명, "1"은 불투명을 의미합니다. opacity 속성은 요소 전체에 적용되며, 속성이 상속되기 때문에 포함하는 자식 요소들에게도 영향을 끼칩니다.

/* 예시 */
div { opacity : 0.5 ; }

# background–color 속성

background-color는 요소의 배경색을 지정하는 속성으로, background-color 속성이 적용되는 범위는 기본적으로 요소의 content 영역과 padding 영역입니다. background-color 속성은 color 속성과 마찬가지로 CSS3에서 지원하는 다양한 색상 값을 사용할 수 있기 때문에 RGBA 형식이나 HSLA 형식으로 투명 효과를 지정할 수 있습니다. 이때 opacity 속성을 사용하여 요소를 투명하게 하는 것과의 차이점은 요소 전체의 투명도가 아닌 배경 색상에 대한 투명도만 지정된다는 점입니다.

/* 예시 */
div { background-color : rgba(255,0,0,0.5) ; }

# background–image 속성

background-image는 요소 박스에 배경 이미지를 지정하는 속성입니다. CSS3에서는 배경 이미지의 속성 값에 콤마(,)로 구분하여 여러 개의 이미지를 지정함으로써 하나의 요소에 다중 배경 이미지를 지정할 수 있게 되었습니다. 또한 여러 개의 배경 이미지를 지정할 때 배경 이미지의 겹치는 순서는 가장 먼저 지정한 이미지가 가장 위쪽에 위치합니다.

CSS3에서 background-image 속성의 가장 큰 변화를 꼽자면 요소 박스에 직접 그레이디언트 효과를 지정할 수 있게 되었다는 점입니다. 이로 인해 이미지 없이 점진적인 색상 표현이 가능하고 확대, 축소 시에도 해상도의 손실 없이 표현할 수 있게 되었습니다.

📌linear–gradient( )

Gradient 효과 중 linear-gradient를 사용하면, 선형 그레이디언트 효과를 지정할 수 있습니다.

/* 형식 */
selector {
 background-image : linear-gradient([angle] ,[color-stop]);
}
1
2
3
4

linear-gradient 지정 시 angle은 ‘각도’를 의미합니다. "deg" 단위의 값이나 "to [키워드]" 또는 0%[백분율]와 같은 형식으로 지정할 수 있습니다. 키워드 "top", "bottom", "left", "right"로 "to top right"와 같이 상하 방향과 좌우 방향을 조합하여 사용할 수도 있습니다.

color-stop은 선형 그레이디언트의 색상 변환 점을 지정하기 위한 옵션으로, 색상 값과 변환 점 위치 두 가지 값을 콤마(,)로 구분하여 여러 개를 지정할 수 있습니다. 변환 점의 위치는 길이 단위 값이나 백분율 값으로 나타내며, 선형 그레이디언트 angle 방향의 위치입니다. color-stop에서 변환 점의 위치 값은 생략할 수 있으며, 변환 점 값이 생략될 경우에는 색상 변환 점이 그레이디언트 angle 방향에 대하여 등 간격으로 배치됩니다.

/* 예시 */
div { 
 background-image : linear-gradient(rgba(0,0,0,0.5), rgba(255,0, 0,0.5), rgba(255,255,255,0.5) ;
}
div {
 background-image : linear-gradient(45deg, #000, #F00 80%, #FFF) ;
}

📌radial–gradient( )

radial-gradient는 원형 그레이디언트 효과를 작성할 수 있는 속성 값 형식입니다.

/* 형식 */
selector {
 background-image : radial-gradient([shape] [size] [postion],[color-stop]) ;
}
1
2
3
4

radial-gradient의 shape는 그레이디언트의 형태를 의미합니다. 사용할 수 있는 형태로는 "circle"(원형)과 "ellipse"(타원)이 있습니다. 두 번째로 size는 원형 그레이디언트의 크기입니다. 그레이디언트의 형태가 "circle(원형)"인 경우에는 하나의 길이 단위 값으로 그레이디언트의 지름을 지정하고, "ellipse"(타원)의 경우에는 2개의 길이 값 또는 백분율 단위 값으로 그레이디언트의 길이를 지정합니다. 또한 size는 키워드로도 지정할 수 있으며, 키워드와 그레이디언트의 크기와의 관계는 다음과 같습니다.

키워드 설명
closest-side 그레이디언트 원점부터 가장 가까운 경계선까지 그레이디언트를 형성합니다.
closest-corner 그레이디언트 원점부터 가장 가까운 모서리까지 그레이디언트를 형성합니다.
farthest-side 그레이디언트 원점부터 가장 먼 경계선까지 그레이디언트를 형성합니다.
farthest-corner 그레이디언트 원점부터 가장 먼 모서리까지 그레이디언트를 형성합니다.
contain closest-side와 동일한 효과로 적용됩니다.
cover farthest-corner와 동일한 효과로 적용됩니다.

postion을 이용하면 원형 그레이디언트의 중심점 위치도 지정할 수 있습니다. "at [X좌표 Y좌표]" 또는 "at [키워드]"의 형식으로 지정할 수 있습니다. 지정하는 방식은 속성 값 앞에 "at"을 표기하는 것 이외에는 background-position 속성을 사용하는 방식과 동일합니다.

color-stop은 선형 그레이디언트와 마찬가지로 원형 그레이디언트의 색상 변환 점 역할을 담당합니다. 색상 변환 점은 색상 값과 변환 점 위치의 두 가지 값을 콤마(,)로 구분하여 여러 개를 지정할 수 있습니다. 변환 점 위치는 길이 단위 값이나 백분율 값으로, 원형 그레이디언트 중심에서부터 반지름까지에 대한 위치입니다. color-stop에서 변환 점 위치 값은 생략할 수 있으며, 변환 점 값이 생략될 경우에는 색상 변환 점이 그레이디언트 반지름에 대하여 등간격으로 배치됩니다.

원형 그레이디언트의 시작점을 설정하는 방법은 CSS3의 스펙에서의 정의와 주요 웹 브라우저의 구현에 약간의 차이가 있습니다. CSS3 스펙상에는 "at" 구문을 사용하는 데 반해 주요 웹 브라우저의 구현은 값의 순서에 따라 처음 오는 2개의 값을 시작점의 좌표로 인식합니다.

/* 예시 */
div { background-image : radial-gradient(circle farthest-corner at center, #FFFFFF 0%, #00A3EF 100%) ; }

/* 형식 */
selector {
 background-image : repeating-linear-gradient([angle], [color-stop]) ;
}

2
3
4

repeating-linear-gradient는 그레이디언트의 크기가 요소를 모두 채우지 못할 경우, 색상 변환을 반복하여 요소 전체를 채우는 선형 그레이디언트를 생성합니다. 사용하는 방법은 linear-gradient와 동일하지만, 마지막 색상 변환 점의 위치를 명시적으로 지정해야 한다는 점이 다릅니다.

/* 예시 */
div { background-image : repeating-linear-gradient(45deg, #000, #fc9c04 20px, #054D4A 40px) ; }
div { background-image : repeating-linear-gradient(#000, #fc9c04 20px, #054D4A 40px) ; }

# background–repeat 속성

background-repeat는 배경 이미지의 반복 여부를 지정하는 속성입니다. 배경 이미지가 여러 개 지정 되었을 경우, background-repeat 속성 값도 콤마(,)로 구분하여 여러 번 지정할 수 있으며, 배경 이미지의 순서대로 반복 스타일을 지정하면 됩니다.

배경 이미지를 가로나 세로 방향으로 반복하고자 할 경우에는 repeat-x나 repeat-x 값을 사용할 수 있으며, no-repeat 값을 지정할 경우에는 배경 이미지가 한 번만 지정됩니다.

CSS3에서는 새롭게 space 값과 round 값이 추가되었으며, space 값과 round 값은 배경 이미지가 반복적으로 나타날 경우 잘려 보이지 않도록 처리합니다.

/* 예시 */
div { background-repeat : no-repeat ; }
div { background-repeat : space, no-repeat, round ; }

# background–attachment 속성

background-attachment는 배경 이미지의 고정 여부를 지정하는 속성입니다. 앞의 background 관련 속성과 마찬가지로 콤마(,)로 구분하여 여러 번 지정할 수 있습니다.

fixed 속성 값의 경우에는 콘텐츠 영역의 스크롤과 관계없이 배경 이미지가 뷰 포트(Viewport)에 고정되고, scroll 속성 값은 배경 이미지가 콘텐츠 영역과 함께 스크롤됩니다. CSS3에서는 새롭게 local 속성 값이 추가되었으며, local 속성 값은 요소 내부에서 스크롤이 발생할 경우 요소 내부의 콘텐츠 영역과 함께 스크롤됩니다.

/* 예시 */
div { background-attachment : fixed ; }
1
2

# background–position 속성

background-position은 배경 이미지의 위치를 지정하는 속성입니다. background-position 속성 역시 앞서 살펴본 background 관련 속성과 마찬가지로 콤마(,)로 구분하여 여러 번 지정할 수 있습니다.

키워드의 경우 top은 세로 방향 값을 "0"으로 지정한 것과 동일하고, left는 가로 방향 값을 "0"으로 지정한 것과 동일합니다. bottom은 세로 방향 값을 "100%"로 지정한 것과 동일하고, right는 가로 방향 값을 "100%"로 지정한 것과 동일합니다.

❇️가로 방향으로 70%, 세로 방향으로 50% 지점에 배경 이미지를 배치한 예

background-position 속성은 가로 방향 위치 값과 세로 방향 위치 값을 함께 지정하지만 하나의 키워드를 지정할 경우, 나머지 하나는 center로 인식합니다.

/* 예시 */
div { div { background-position : center ; }
div { background-position: 70% 50% ; }

# background–clip 속성

background-clip은 배경 속성이 적용되는 영역을 지정하는 속성입니다. CSS2.1에서 배경 속성이 적용되는 부분은 콘텐츠 영역과 padding 영역이었습니다. 그러나 CSS3에서는 배경 속성이 적용되는 영역을 직접 지정할 수 있게 되었습니다. 배경 이미지가 여러 개 지정되었을 경우, background-clip 속성의 적용 영역 설정도 콤마(,)로 구분하여 여러 번 지정할 수 있으며, 배경 이미지의 순서대로 background-clip 속성이 지정됩니다.

border-box 속성 값은 기본 값으로, 콘텐츠 영역, padding 영역, border 영역까지 배경 이미지가 표시됩니다. 그리고 padding-box와 content-box 속성 값은 각각 padding 영역 또는 콘텐츠 영역에만 배경 이미지가 표시됩니다.

/* 예시 */
div { background-clip : content-box ; }
1

# background–origin 속성

background-origin은 배경 이미지의 시작점을 지정하는 속성으로, CSS2.1에서 배경 이미지의 시작점은 padding 영역의 좌측 상단이었습니다. 그러나 CSS3에서는 배경 이미지의 시작점을 직접 지정할 수 있게 되었습니다. background-origin 속성 역시 다른 background 관련 속성과 마찬가지로 배경 이미지가 여러 개 지정되었을 경우, 배경 이미지의 시작점도 콤마(,)로 구분하여 여러 번 지정할 수 있으며, 배경 이미지의 순서대로 background-orign 속성이 지정됩니다.

background-clip 속성과 마찬가지로 border와 padding 그리고 content 영역으로 구분하여 배경 이미지가 시작되는 지점을 지정할 수 있습니다.

/* 예시 */
div { background-origin : content-box ; }

# background–size 속성

background-size 역시 CSS3에 새롭게 추가된 속성으로, 배경 이미지의 크기를 지정하는 속성입니다. 사용할 수 있는 속성 값은 일반적인 길이(length) 단위인 백분율(percentage)을 통해 배경 이미지의 크기를 직접 지정할 수 있으며, contain이나 cover와 같은 키워드로도 지정할 수 있습니다. 이때 contain 키워드는 배경 이미지의 원래 비율을 유지한 채 요소 박스 영역 전체에 배경 이미지가 가득 차도록 하고, cover는 배경 이미지의 원래 비율을 무시하고 요소 박스 영역 전체에 배경 이미지가 가득 차도록 합니다.

background-size 속성 역시 배경 이미지가 여러 개 지정되었을 경우 콤마(,)로 구분하여 여러 번 지정할 수 있으며, 배경 이미지의 순서대로 background-size 속성이 지정됩니다.

/* 예시 */
div { background-size : 50px 50px ; }
div { background-size : 50% 50% ; }
div { background-size : cover ; }

# background 속성

background 속성은 앞서 살펴본 배경 관련 속성을 일괄적으로 지정하는 속성입니다.

background는 배경 이미지 및 위치와 크기 반복 스타일, 고정 여부 및 배경 색상 등을 일괄적으로 적용할 수 있는 대표 속성입니다. 일괄적으로 지정하는 값이 여러 개일 경우 배경 이미지별로 콤마(,)로 구분하여 여러 번 지정하는 것이 가능하지만, 배경 색상은 한 가지 색만 지정할 수 있습니다. 이때 배경 이미지가 여러 번 지정될 경우, 배경 색상은 가장 마지막에 지정한 배경 이미지와 함께 지정해야 합니다.

/* 예시 */
div { background : url(images/tree.png) 100% 100% /100px 130px no-repeat scroll, 
                   url(images/rabbit.png) 70px 80px /50px 50px no-repeat scroll #FFFFCC;
}

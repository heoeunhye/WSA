# CSS 박스 모델

CSS는 요소를 표시하기 위해 요소별로 사각형의 박스를 생성합니다. CSS3에서는 속성이 다양해졌지만 요소를 사각형 박스로 표시하는 것은 크게 다르지 않습니다. HTML 요소 박스의 크기는 콘텐츠 크기에 안쪽 여백인 padding과 테두리인 border 그리고 바깥쪽 여백인 margin에 의해 결정됩니다.

❇️박스가 실제 화면에서 차지하는 크기를 결정하는 요인

  • content : 요소의 콘텐츠가 표시되는 영역
  • padding : content 영역과 border 사이의 안쪽 여백
  • border : padding 영역과 margin 사이의 테두리
  • margin : border를 기준으로 다른 요소와의 바깥쪽 여백

이렇게 HTML 요소 박스는 다양한 속성에 의해 결정되며, 이를 ‘CSS 박스 모델(Box Model)’이라고 합니다. 이러한 개념은 CSS2.1에서도 동일하게 제공되었습니다. 기본기를 다지기 위해 이번에는 Basic Box Model 관련 속성에 대해 살펴보겠습니다.

# display 속성

display는 HTML 요소의 표현 방식을 지정하는 속성으로, HTML 요소는 각각 기본으로 지정된 display 값이 존재하며, 그 값도 매우 다양합니다.

display 속성 값이 "inline"으로 지정된 요소는 자신의 전후로 줄바꿈을 만들지 않으며, 포함하는 요소가 너비에 의해 자동 줄바꿈됩니다. 요소의 폭은 콘텐츠에 따라 결정되며, width 속성으로 요소의 너비를 직접 지정할 수는 없습니다. "block"으로 지정된 요소는 기본적으로 부모 요소의 너비 전체를 채우며, 요소 박스 전후로 줄바꿈이 생깁니다. 그리고 "inline-block"으로 지정된 요소는 "inline"과 같이 요소 전후로 줄바꿈이 생기지는 않지만, block 요소처럼 width 속성으로 요소의 너비를 직접 지정할 수 있습니다.

display 속성은 이 밖에도 화면에 요소를 표시하지 않는 "none" 값을 비롯하여 요소의 형제 요소가 block 요소가 아니면 block 요소로 정의되고, block 요소이면 inline 요소로 정의되는 "run-in" 및 목록 요소로 보여지도록 하는 "list-item", 테이블로 표시하는 table 관련 값과 루비 요소로 표현하는 "ruby" 등 다양한 값을 지정할 수 있습니다.

/* - 예시 */
span { display : block ; }
div { display : inline ; }
span { display : inline-block ; }

# padding 속성

padding은 content 영역과 border 사이의 안쪽 여백을 지정하는 속성으로, 속성 값은 1에서 4개까지 입력할 수 있으며, 입력 값의 개수에 따라 적용되는 방향이 다릅니다.

padding 속성은 padding-[방향]의 상세 속성을 이용하여 방향별로 지정할 수 있으며, 이때는 하나의 속성 값만을 사용합니다. 사용할 수 있는 속성 값은 길이 단위 값으로, 음수 값은 사용할 수 없습니다. 만약 속성 값에 백분율을 사용할 경우, 상위 요소의 크기를 참조하여 비율로 결정합니다. 백분율 역시 음수 값은 사용할 수 없습니다.

/* - 예시 */
div { padding : 10px ; }
div { padding : 10px 20px ; }
div { padding : 10px 20px 30px ; }
div { padding : 10px 20px 30px 40px ; }
div {
 padding-top : 10px ;
 padding-right : 20px ;
 padding-bottom : 30px ;
 padding-left : 40px ;
}

# border 속성

border는 요소 박스의 테두리를 지정하는 속성입니다. border 속성은 border-[방향] 등으로 박스의 특정 위치만을 지정할 수 있습니다. 또한 border-style, border-width, border-color 등의 세부 속성을 사용하여 테두리 선의 모양과 굵기 및 색상을 지정할 수도 있습니다.

border 관련 속성은 CSS3에서 많은 기능이 추가된 부분이기도 합니다. CSS3의 모듈에서도 박스 모델과 별도의 모듈로 분리되어 있습니다. 그러므로 여기서는 간단히 border 속성과 함께 개념만 다루고 border-color, border-style, border-width 속성을 포함한 다양한 테두리 관련 속성은 뒤에서 자세히 살펴보겠습니다.

/* - 예시 */
div { border : 1px solid #ccc ; }
div { border-width : 1px ; }
div { border-style : solid ; }
div { border-color : #ccc ; }

# margin 속성

margin 속성은 border를 기준으로 다른 요소와의 바깥쪽 여백을 지정하는 속성으로 padding 속성과 사용 방법이 동일합니다.

margin 속성은 padding 속성과 마찬가지로 1개에서 4개까지 입력할 수 있으며, 입력 값의 개수에 따라 적용되는 방향이 다릅니다.

margin 속성은 padding 속성과 달리 속성 값에 음수 값을 사용할 수 있습니다. 그리고 margin-[방향]의 상세 속성을 이용하여 방향별로 지정할 수도 있으며, 이때는 하나의 속성 값만 사용할 수 있습니다.

margin의 특성 중에 상하로 인접한 박스의 display 속성 값이 "block"인 경우, 마진 겹침(Margin Collapsing) 현상이 발생합니다.

앞의 그림과 같이 A 요소의 margin-bottom 속성 값이 "10px"이고, B 요소의 margin-top 속성 값이 20px"일 때 두 요소 사이의 margin은 20px로 적용됩니다. margin 속성의 이러한 겹침 현상은 레이아웃 설계 시 많은 혼란을 야기할 수 있습니다. 때로는 버그처럼 이해하는 경우도 있는데, 수직으로 선형화 된 박스의 간격을 균등하게 지정할 수 있는 합리적인 방법이라 할 수 있습니다.

/* - 예시 */
div { margin : 10px ; }
div { margin : 10px 20px ; }
div { margin : 10px 20px 30px ; }
div { margin : 10px 20px 30px 40px ; }
div {
 margin-top : 10px ;
 margin-right : 20px ;
 margin-bottom : 30px ;
 margin-left : 40px ;
}

# width, height 속성

width와 height는 요소의 너비와 높이를 지정하는 속성입니다. display 속성 값이 "inline"인 요소에는 적용되지 않습니다.

  • auto : 기본 값으로 display 속성 값이 "block"인 경우 width 속성은 기본적으로 화면 너비 전체를 차지하며, height 속성은 콘텐츠 높이에 따라 자동으로 조절됩니다.
  • length : 길이 단위 값으로 적용되는 방식은 box-sizing 속성 값에 따릅니다. height 속성의 경우 콘텐츠 높이가 지정된 값보다 크더라도 요소의 높이는 변경되지 않으며, 음수 값은 지정할 수 없습니다(box-sizing 속성은 User Interface 파트 참조)
  • percentage : 자신을 포함한 요소의 값을 참조하여 비율로 결정합니다. height 속성의 경우 length처럼 콘텐츠 높이로 계산된 값이 지정된 값보다 크더라도 요소의 높이는 변경되지 않으며, 음수 값은 지정할 수 없습니다.
/* - 예시 */
div { width : 100px ; }
div { width : 50% ; }
div { height : 100px ; }
div { height : 20% ; }

# min–width, min–height, max–width, max–height 속성

min-width, min-height, max-width, max-height 속성은 요소의 최소 및 최대 너비와 높이를 지정하는 속성입니다. width 및 height 속성과 사용 값이 동일합니다.

/* - 예시 */
div { min-width : 200px ; }
div { max-width : 50% ; }
div { min-height : 200px ; }
div { max-height : 50% ; }

# visibility 속성

visibility는 HTML 요소에 의해 만들어진 박스의 화면상 표시 여부를 지정하는 속성으로, 기본 값은 화면에 요소를 표시하는 "visible"입니다. visibility 속성 값을 "hidden"으로 지정할 경우에는 요소를 화면에서 숨깁니다. visibility:hidden은 화면상에 요소를 표시하지 않는 display:none과 달리 요소의 원래 영역은 유지하게 됩니다. "collapse"는 테이블 내부 객체로, 테이블의 행, 행 그룹, 열, 열 그룹에 적용되며 "hidden"과 비슷합니다.

/* - 예시 */
div { visibility : hidden ; }

# overflow 속성

overflow는 콘텐츠가 요소의 콘텐츠 영역을 벗어나는 경우의 처리 방법을 지정하는 속성으로, "visible"은 오버 플로우된 콘텐츠를 그대로 표시합니다. 이때 오버 플로우된 콘텐츠의 영역은 요소의 크기에 반영되지 않습니다. "hidden"의 경우 오버 플로우된 콘텐츠를 숨기고 "scroll"은 요소 박스에 스크롤 바를 생성합니다. "auto" 속성 값의 경우 사용자 도구의 기본 설정 값대로 작동하도록 합니다.

overflow 속성을 방향별로 지정하기 위한 상세 속성인 overflow-x나 overflow-y를 사용할 수 있습니다.

/* - 예시 */
div { overflow : auto; }

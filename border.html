# 테두리 관련 속성

border는 요소 박스의 테두리를 의미합니다. CSS는 요소를 사각형의 박스로 표시하기 때문에 border 역시 사각형으로 그려집니다. 그러나 기존 CSS2.1에서는 사각형으로 표시할 수밖에 없었던 border를 CSS3에서는 다양한 방법으로 표현할 수 있게 되어 활용 폭이 크게 증가하였습니다. 이번에는 CSS3에서 많은 부분이 추가되거나 변화한 테두리 관련 속성에 대해 살펴보겠습니다.

# border–color 속성

border-color는 요소 박스의 테두리 선 색상을 지정하는 속성입니다. border-color 속성 값을 지정하지 않은 경우에는 기본 색상 값이 currentcolor, 즉 요소의 color 속성 값으로 지정됩니다.

테두리 선의 색상 값은 1개에서 4개까지 지정할 수 있으며, 값의 개수별로 적용되는 방향이 다릅니다. 세부 속성인 border-[방향]-color 속성을 적용하면 각 방향별로도 테두리 색상을 지정할 수 있습니다.

/* 예시 */
div { border-color : rgba(255,0,0,0.7) ; }
div { border-color : #f00 #0f0 #00f #000 ; }

# border-style 속성

border-style은 요소 박스 테두리의 선 스타일을 지정하는 속성입니다. border-color 속성과 마찬가지로 값은 1개에서 4개까지 지정할 수 있으며, 값의 개수별로 적용되는 방향이 다릅니다. 세부 속성인 border-[방향]-style을 적용하면 각 방향별로도 지정할 수 있습니다.

/* 예시 */
div { border-style : solid ; }
div { border-style : solid dotted double dashed ; }

2
3

# border-width 속성

border-width는 요소 박스 테두리의 선 굵기를 지정하는 속성입니다. border-width 속성 역시 굵기에 해당하는 속성 값을 1개에서 4개까지 지정할 수 있으며, 세부 속성인 border-[방향]-width를 적용하면 각 방향별로도 지정할 수 있습니다.

border-width 속성 값인 키워드의 경우 "thin", "medium", "thick"에 대한 명확한 수치는 규정되어 있지 않으며, 각 값이 상대적으로 사용자 에이전트에 의해 결정됩니다.

/* 예시 */
div { border-width : 1em ; }
div { border-width : 5px 10px 10px 5px ; }

# border-radius 속성

border-radius는 요소 박스의 테두리 선을 둥근 모서리 형태로 지정하는 속성입니다. 이때 둥근 모서리의 길이는 가로 방향과 세로 방향의 값을 "슬래시(/)"로 구분하여 지정할 수 있으며, 각 방향별로 값을 1개에서 4개까지 지정할 수 있습니다. 세부 속성인 border-[세로 방향]-[가로 방향]-radius를 적용하면 방향 별로 각각 둥근 형태를 지정할 수 있습니다.

/* 예시 */
div { border-radius : 10px ; }
div { border-radius : 5px 10px 15px 20px ; }
div { border-radius : 5px 10px 15px 20px / 10px 5px 5px 10px ; }

# border–image–source 속성

border-image-source는 요소 박스 테두리에 표시될 이미지를 지정하는 속성입니다. border-imagesource 속성이 적용된 요소는 order-style 대신 border-image-source에서 지정한 이미지가 표시되며, 기본적으로 border-width 속성에서 지정된 굵기 영역에 이미지가 나타납니다.

/* 예시 */
div { border-image-source : url(images/border_image.png) ; }
1
2

# border–image–slice 속성

border-image-slice는 테두리에 지정할 이미지가 요소 박스 상하좌우 테두리 영역에 보여질 크기를 지정하는 속성입니다. 이때 number 속성 값의 테두리에 표시될 이미지 부분을 단위가 없는 수치 값으로 지정하면 비트맵 이미지에서는 px 단위로, 벡터 이미지에서는 좌표 단위로 인식하며 값은 1개에서 4개까지 지정할 수 있습니다.

fill 키워드를 사용할 경우, 이미지의 가운데 부분도 표시되도록 지정합니다. border-image-slice 속성 값에는 음수 값을 사용할 수 없으며, 만약 이미지의 크기보다 큰 값이 지정된 경우에는 100%로 간주하여 표시됩니다.

/* 예시 */
div { border-image-slice : 30 15 20 10 ; }
1
2

# border–image–repeat 속성

border-image-repeat는 요소 박스에 지정된 이미지가 반복되는 스타일을 지정하는 속성입니다. 이때 속성 값 stretch는 이미지를 테두리 크기만큼 늘려 표시하고, repeat는 반복하여 표시합니다. 그리고 round는 이미지를 분할한 규칙대로 반복하며, 반복 횟수가 정확히 일치하지 않을 경우에는 이미지의 크기를 조정하여 표시합니다. 마지막으로 space는 이미지를 분할한 규칙대로 반복하며, 반복 횟수가 정확히 일치하지 않을 경우에는 반복되는 이미지 사이의 간격을 배분하여 표시합니다.

/* 예시 */
div { border-image-repeat : repeat ; }
div { border-image-repeat : repeat stretch ; }

# border–image–width 속성

border-image-width는 각 방향별로 이미지 표시 영역의 길이를 지정하는 속성입니다. number 속성 값의 경우 border-width 속성에 지정된 수치를 곱하여 요소 박스의 테두리 영역의 길이를 결정합니다. length, percentage, number는 속성 값을 1개에서 4개까지 지정할 수 있으며, 값의 개수별로 속성이 지정되는 방향이 다릅니다. 그리고 auto 속성 값을 지정하며, border-image-slice 속성에 지정된 값과 동일한 크기로 적용됩니다.

border-image-width 속성의 기본 값은 1로, border-image-width 속성을 지정하지 않을 경우에는 border-width 속성과 같은 값을 가지게 되므로, 이미지는 border-width 속성의 너비로 표시됩니다. border-image-width 속성 값을 지정하여 border-image-width 속성 값이 border-width 속성 값보다 커질 경우에는 이미지 표시 영역이 콘텐츠 방향으로 늘어나게 됩니다. 또한 border-image가 padding 영역이나 content 영역을 침범하더라도 두 영역의 레이아웃에는 영향을 끼치지 않습니다.

/* 예시 */
div { border-image-width : 50px ; }

# border–image–outset 속성

border-image-outset은 테두리에 지정한 이미지가 border 영역에서 벗어나 요소 박스 외부에 표시되는 이미지 영역을 지정하는 속성입니다. 이렇게 확장된 영역은 스크롤되지 않으며, 마우스 이벤트의 영향을 받지도 않습니다. 값은 1개에서 4개까지 지정할 수 있습니다.

/* 예시 */
div { border-image-outset : 20px ; }
1
2

# border–image 속성

border-image는 요소 박스의 테두리 이미지 지정과 관련된 속성을 일괄 적용하는 대표 속성입니다.

/* 예시 */
div { border-image : url(images/border_image.png) 30 round ; }

# box–decoration–break 속성

box-decoration-break는 요소 박스가 페이지나 단으로 인해 나누어졌거나 인라인 요소가 자동 줄바꿈 때문에 나누어졌을 경우, border나 padding을 추가할 것인지의 여부를 지정하는 속성입니다. 사용 가능한 속성 값으로는 나누어진 부분에 border와 padding을 추가하지 않는 slice와 추가하는 clone이 있습니다.

/* 예시 */
div { box-decoration-break : clone ; }
1
2

# box–shadow 속성

box-shadow는 요소 박스의 그림자를 지정하는 속성입니다. text-shadow 속성 값과 마찬가지로 처음 등장하는 2개의 길이 값은 그림자의 가로 및 세로 방향의 위치를 의미하고, 세 번째 길이 값은 blur의 반 지름을 의미합니다. 그리고 text-shadow 속성에는 없었던 네 번째 길이 값은 spread 반경으로, 그림자의 확장 영역을 지정할 수 있습니다. 또한 inset 키워드를 사용하여 박스의 안쪽으로 그림자를 표현할 수 있으며, 콤마(,)로 구분하여 여러 개의 박스 그림자를 지정할 수도 있습니다.

/* 예시 */
div { box-shadow : inset 5px 5px 5px 5px rgba(0, 0, 0, 0.5) ; }
